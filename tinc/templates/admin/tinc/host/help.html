Tinc client installation help

{% endblock %} {% block content %}

The following instructions are for a debian-like linux distribution.

1. Install dependencies

apt-get install tinc ip

2. Create and configure the tinc network

mkdir /etc/tinc/{{ net_name }}/
echo "Name = {{ host.tinc.pk }}" > /etc/tinc/{{ net_name }}/tinc.conf
echo "ConnectTo = server" >> /etc/tinc/{{ net_name }}/tinc.conf
mkdir /etc/tinc/{{ net_name }}/hosts
echo "Subnet = {{ host.tinc.subnet }}" > /etc/tinc/{{ net_name }}/hosts/{{ host.tinc.pk }}
echo '
{% for addr in host.tinc.connect_to.all %}Address = {{ addr }}{% endfor %}
Port = 655
Subnet = {{ server.tinc.subnet }}

{{ server.tinc.pubkey }}'> /etc/tinc/{{ net_name }}/hosts/server
echo '#!/bin/sh
ip -6 link set "$INTERFACE" up mtu 1400
ip -6 addr add {{ host.tinc.subnet|cut:"0/64" }}2/48 dev "$INTERFACE"' > /etc/tinc/{{ net_name }}/tinc-up

echo '#!/bin/sh
ip -6 addr del {{ host.tinc.subnet|cut:"0/64" }}2/48 dev "$INTERFACE"
ip -6 link set "$INTERFACE" down' > /etc/tinc/{{ net_name }}/tinc-down
# Make the scripts executable:
chmod a+rx /etc/tinc/{{ net_name }}/tinc-{up,down}

# Generate the key pairs of your host:
tincd -n {{ net_name }} -K

3. Upload the key to the server

Upload your public key on your {{ host.description|capfirst }} host page.

You can found the key at the end of /etc/tinc/{{ net_name }}/hosts/{{ host.tinc.pk }} file.

4. Start tinc daemon

echo {{ net_name }} >> /etc/tinc/nets.boot
/etc/init.d/tinc restart
# Try if your connection is working by ping the server
ping6 {{ server.tinc.subnet|cut:"/128" }}
